## 🌌 Sobre o projeto

Este é meu portfólio pessoal, desenvolvido para apresentar minha trajetória como desenvolvedor, meus principais projetos, experiências profissionais, certificações e habilidades.

Todo o conceito visual foi inspirado no espaço, utilizando estrelas, constelações, partículas e animações para criar uma experiência única de navegação.

Mais do que um site de apresentação, este projeto representa minha forma de desenvolver interfaces modernas, organizadas e com foco em experiência do usuário.

---

## ✨ Principais recursos

- 🌙 Dark Mode e White Mode
- 🚀 Hero totalmente animada
- ✨ Cursor personalizado
- 🌌 Galáxia interativa de Skills

- 🏆 Área de certificações em formato de constelação

## 🚀 Executando localmente

Clone o projeto

```bash
git clone https://github.com/Paulo-Vitor-dev/My-portfolio-.git
```

Entre na pasta

```bash
cd My-portfolio-
```

Instale as dependências

```bash
npm install
```

Execute

```bash
npm run dev
```

O projeto estará disponível em

```
http://localhost:3000
```
